# GastoClaro

GastoClaro es una aplicación personal orientada al control de flujo mensual, vencimientos, deudas, ingresos programados y registro de pagos.

## Estructura del proyecto

- `docs/` documentación funcional y técnica
- `gastoclaro-api/` backend Laravel
- `gastoclaro_app/` app Flutter

## Documentación

- Enfoque personal del MVP
- Modelo técnico del MVP
- Roadmap técnico de implementación

## Objetivo de la versión 1

Construir una herramienta personal real para:

- ver vencimientos del mes
- entender presión financiera mensual
- ordenar deudas
- registrar ingresos programados
- registrar pagos reales

## Stack

- Backend: Laravel
- Frontend móvil/web: Flutter
- Base de datos objetivo: PostgreSQL
